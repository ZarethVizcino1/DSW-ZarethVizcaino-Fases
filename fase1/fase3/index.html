Introducción

El sitio esta siendo diseñado de forma educativa para la formacion diseño de sitios web para dinamizar las estrategias de aprendizaje. Una estrategia la constituyen los Objetos virtuales de aprendizaje. Estos recursos son desarrollados por empresas que diseñan contenidos multimediales e interactivos, basados en HTML5, CSS3 entre otras tecnologías. En esta pagina se relaizara una pequeña conceptualizacion del que es sintaxis y los selectores CSS y la interacción humano-ordenador ademas, se encontrara informacion y podcast referentes al los temas tratados.

El lenguaje de etiqeutado HTML permite darle formato a el contido de las paginas web, designando mediante cada etiqueta distintos elementos que le permiten al desarrollador determinar la finalidad de cada elemento y el formato que deben tener.
